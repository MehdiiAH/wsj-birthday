# Jump Birthday 🗞️

> Which issue of Weekly Shōnen Jump was on newsstands when you were born?

---

## Features

- 📅 Date picker with shareable URL (`?d=YYYY-MM-DD`)
- 🖼️ High-res cover image with lightbox (Comic Vine)
- 📚 Double-issue and bi-weekly detection
- 🔗 One-click link copy + Twitter/X share
- 🌍 7 languages: EN, FR, ES, JA, PT, DE, IT

---

## Stack

Next.js 15 · next-intl · Tailwind CSS 3 · React 19 · TypeScript
