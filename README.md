# Jump Birthday 🗞️

> Find out which issue of Weekly Shōnen Jump was on newsstands the day you were born.

---

## Features

- **2,832 issues** covered — August 1968 through 2026
- **7 languages** — French, English, Spanish, Japanese, Portuguese, German, Italian
- **Shareable URLs** —  pre-fills the date on load
- **One-click share** — copy link or post directly to X/Twitter
- **Smart coverage logic** — handles double issues and the pre-1969 biweekly period
